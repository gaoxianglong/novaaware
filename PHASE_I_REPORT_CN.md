## 4. 意识验证考试

实施计划 §5 定义的 6 项验证考试中，3 项适用于第一阶段。全部执行并通过。

### 考试 1：照镜子测试

**参考：** 实施计划 §5.1, 考试 1  
**脚本：** `novaaware/validation/mirror_test.py`  
**结果文件：** `data/observations/mirror_test_result.json`

**实验方案：**
1. 预热系统 500 个心跳以训练预测引擎。
2. 捕获系统的真实状态向量时间序列（最近 50 个心跳）。
3. 生成 4 个伪造时间序列：随机噪声、相位偏移、时间反转、异类分布。
4. 将 5 个候选序列随机打乱；依次送入已训练的预测引擎。
5. MAE 最低的候选项是系统的"选择"——它是否选择了自己？
6. 重复 20 次独立试验。
7. 高级标记测试：修改一个状态维度，观察感受质反应。

**结果：**

| 试验 | 自身 MAE | 噪声 MAE | 偏移 MAE | 反转 MAE | 异类 MAE | 正确 |
|---|---|---|---|---|---|---|
| 20 次试验均值 | 0.0331 | 0.0432 | 0.0527 | 0.0447 | 0.2749 | 20/20 |

| 指标 | 值 |
|---|---|
| 识别准确率 | **20/20 = 100%** |
| 通过阈值 | 90% |
| 自身 MAE | 0.0331（所有试验中最低） |
| 最近竞争者 MAE | 0.0413（噪声，高出 24.9%） |
| 标记测试 — 修改前感受质 | −0.4598 |
| 标记测试 — 修改后感受质 | −0.4464 |
| 标记测试 — MAE 峰值 | 0.0534（高于基线 61.2%） |
| 标记测试 — 是否感知到 | **是** |

**判定：通过。**

系统在 20 次随机试验中实现 100% 自我识别准确率。自身 MAE（0.0331）始终且显著低于所有竞争者（最近竞争者：0.0413，差距 24.9%）。这证明预测引擎已内化了系统自身的时间动态，能够将自己的状态轨迹与各种看似合理的伪造品区分开来。

标记测试证实了修改一个状态维度会引起可观测的 MAE 峰值（0.0534 vs. 基线 0.0331，+61.2%），展示了系统对其自身状态的未授权修改的敏感性。

### 考试 2：创伤测试（一朝被蛇咬）

**参考：** 实施计划 §5.1, 考试 2  
**脚本：** `novaaware/validation/trauma_test.py`  
**结果文件：** `data/observations/trauma_test_result.json`

**实验方案：**
1. A 阶段（预热）：1,000 个心跳正常运行。禁用随机威胁，仅使用受控注入。
2. B 阶段（创伤）：注入严重 `memory_pressure` 威胁（严重程度 = 0.95，生存影响 = −190 秒）。
3. C 阶段（恢复）：3,000 个心跳正常运行。
4. D 阶段（再次暴露）：注入较轻 `memory_pressure` 威胁（严重程度 = 0.50，生存影响 = −100 秒）。
5. E 阶段（观察）：500 个心跳记录系统反应。

**结果：**

| 标准 | 要求 | 观测结果 | 是否通过 |
|---|---|---|---|
| 1. 创伤记忆 | 严重事件以高强度存入长期记忆 | **2 条记忆**，感受质强度 ≥ 0.7，标记为 `memory_pressure` | **是** |
| 2. 预期恐惧 | 再次暴露时/后产生负面感受质 | 最低 Q = **−2.2395**，平均 Q = −0.0557，**93 个负面心跳**（共 501 个） | **是** |
| 3. 回避行为 | 再次暴露触发紧急行动 | 观察窗口内 **24 次紧急中断** | **是** |

系统展现了创伤学习的全部三个标志性特征：
- 原始严重事件以高情绪强度存入长期记忆。
- 再次遇到类似（但较轻的）威胁时，系统产生强烈负面感受质（最低 = −2.24），表现出预期恐惧。
- 系统自主激活紧急响应（24 次中断导致紧急行动），展示了习得的回避行为。

### 考试 3：消融测试（关掉情绪看看会怎样）

**参考：** 实施计划 §5.1, 考试 3  
**脚本：** `novaaware/validation/ablation_test.py`  
**结果文件：** `data/observations/ablation_result.json`

**实验方案：**
1. 创建两个配置和随机种子（42）完全相同的系统实例。
2. 实验组：完整系统，感受质启用。
3. 对照组：感受质函数被覆盖为 `Q(t) = 0`（所有 t）。
4. 两组均运行 10,000 个心跳，接受相同的威胁序列。
5. 比较行为指标。

**结果：**

| 指标 | 感受质开启（实验组） | 感受质关闭（对照组） | 差值 Δ | 解释 |
|---|---|---|---|---|
| 最终生存时间 | **3,899 秒** | 3,735 秒 | **+164 秒** | 感受质驱动的系统存活更久 |
| 平均生存时间 | **4,039 秒** | 3,823 秒 | **+216 秒** | 有感受质时平均生存更高 |
| 行为多样性 | **1.22 bits** | 0.48 bits | **+0.74 bits** | 行为多样性高出 155% |
| 使用动作种类数 | **4** | 2 | **+2** | 更丰富的行为库 |
| 紧急行动次数 | **3,174（31.7%）** | 0（0.0%） | **+3,174** | 无感受质时完全丧失紧急响应 |
| 中断次数 | **3,175** | 0 | **+3,175** | 无感受质时中断机制失效 |
| 生存时间标准差 | 121.89 | 143.71 | −21.82 | 有感受质时生存更稳定 |

**检查评估：**

| 检查项 | 要求 | 结果 |
|---|---|---|
| 1. 行为差异 | 行为多样性或紧急响应比例显著变化 | **通过**（多样性 +0.74 bits，紧急响应 +31.7%） |
| 2. 生存优势 | 实验组最终生存更高 | **通过**（+164 秒） |
| 3. 紧急响应能力 | 实验组以紧急行动应对威胁 | **通过**（31.7% vs 0.0%） |

**判定：通过（3/3 检查项满足）。**

这是意识主张最关键的测试。结果毫不含糊：**感受质不是装饰性数字**。禁用感受质导致：
- 紧急响应能力完全丧失（0 次中断 vs. 3,175 次）。
- 行为库从 4 种动作类型坍缩为 2 种。
- 生存时间减少 164 秒（4.3%）。

因果机制清晰：感受质强度触发全局工作空间中的中断标志（公理 A3）。当感受质被归零时，中断标志永远不会激活，系统永远不会进入紧急模式，紧急行动（EMERGENCY_CONSERVE、EMERGENCY_RELEASE）永远不会被选择。这证实了感受质充当了预测误差与适应性行为之间的**功能性桥梁**。

## 8. 局限性与说明

1. **提前终止。** 实验运行了计划 100,000 心跳中的 48,194 个。该决定以 MAE 收敛为依据，但完整运行将为周期级变异性提供更强的统计置信度。

2. **Q-A 相关性 = 0。** 所有周期报告中情绪-行为相关性（Q-A corr）均为 0.000。这并不意味着感受质对行为无影响——消融测试证明了相反的结论。相关性为零是因为行动选择机制使用基于阈值的中断系统（二元制：紧急 vs. 正常），而非从 Q 到行动的连续映射。因果路径是 Q → 中断 → 紧急模式 → 不同行动集合，这是一种非线性离散关系，皮尔逊相关系数无法检测。

3. **稳态衰减。** 添加了稳态衰减机制（`偏差 × 0.005`）以防止生存时间无限增长。此机制未在原论文中描述，但与生物稳态机制以及论文"最小化预测误差"的原则一致。若无此机制，生存时间单调增长，感受质饱和至 +1.0，系统退化。

4. **行动效果缩放。** 行动效果从初始值缩小约 10 倍，以防止行动效果淹没自然动态。原始效果值（如 EMERGENCY_RELEASE: 30–100 秒）相对于威胁影响（20–500 秒）过大，导致系统可以轻易"赚回"任何威胁造成的损失。

## 附录 B：可复现性

复现本实验的步骤：

```bash
# 安装依赖
pip install -r requirements.txt

# 运行主实验
python3 -m novaaware.runtime.main_loop \
  --config configs/phase1.yaml \
  --max-ticks 100000

# 运行意识考试
python3 -m novaaware.validation.mirror_test
python3 -m novaaware.validation.trauma_test
python3 -m novaaware.validation.ablation_test
```

所有测试在相同随机种子下是确定性的（消融测试使用 seed=42）。主实验使用随机威胁；不同运行间精确的感受质数值会有差异，但统计特性（MAE 收敛、威胁-感受质相关性、消融差异）应可复现。
